Privacy Policy

Gradintel — Canvas Sync Chrome Extension  ·  Effective: March 8, 2026

We do not sell, rent, or share your personal data or academic records with any third party for advertising or commercial purposes.

1. Overview

Gradintel operates the Gradintel Canvas Sync Chrome Extension (the "Extension"). This Privacy Policy explains what information the Extension collects, how it is used, and your rights regarding that information.

The Extension is designed so that your academic data stays under your control at all times.

2. Information We Collect

The Extension collects only the minimum data necessary:

Data CollectedWhyStored Where
Email & session token Authenticate your Gradintel account Browser storage + Supabase
Canvas domain URL Know which Canvas to connect to Browser storage only
Course & grade data Import academic records into Gradintel Your Gradintel account
Canvas session cookies Authenticate Canvas API requests without a token Never stored — memory only

We do NOT collect:

Canvas username or password Browsing history IP address or location Data from other websites Data when Extension is idle
3. Data Storage & Security
Browser Storage

Session token (encrypted by Chrome) & Canvas domain URL. Never sent to any external server.

Supabase (Your Account)

Course names, grades, credits you choose to import. Protected by row-level security — only you can access it.

🔒 Canvas session cookies are never stored. Used in-memory only during an active sync and discarded immediately after. Your Canvas login credentials are never accessed by the Extension.
4. Canvas API Usage
→
Uses your existing browser session — same auth as browsing Canvas normally
→
Does not create, store, or transmit any Canvas access tokens
→
Initiated only when you actively press the "Sync" button
→
Subject to your university's Canvas terms of service and acceptable use policies
5. Your Rights
✓ Access all data in your Gradintel account at any time via the website
✓ Delete your imported grade data at any time from within Gradintel
✓ Uninstall the Extension at any time — this automatically removes all locally stored data
✓ Request full account deletion by contacting us below
