# 📱 Gradintel Android

> Track · Predict · Improve — now on Android.

A native Android wrapper for [Gradintel](https://gradintel.anshulraman.com) — the GPA tracker that tells you exactly what you need to score on every remaining exam.

---

## What is this?

This is the Android app version of Gradintel. It wraps the full Gradintel web experience in a native Android shell — complete with a branded splash screen, dark theme, pull-to-refresh, offline detection, and hardware back navigation.

Everything syncs to your Gradintel account. Same data, same AI assistant, same Canvas integration — on your phone.

---

## ✨ App Features

- Branded splash screen with animated logo
- Dark theme matching the website exactly — `#07070e` background, purple/pink accents
- Full Gradintel web app rendered in a hardware-accelerated WebView
- Pull to refresh
- Offline detection with friendly retry screen
- Hardware back button navigation
- Edge-to-edge display — notch and punch-hole aware
- Compatible with Android 7.0 and above
